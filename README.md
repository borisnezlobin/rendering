This is where I built my rasterizer and raytracer. The raytracer was modelled off of a book titled _Raytracing in One Weekend_ (it took longer than one weekend).

I built the rasterizer from first principles in camera projection physics, with no external references (except for loading OBJ files) or libraries other than a linear algebra utility.

[Read my barebones writeup!](https://www.borisnezlobin.com/projects/rendr)
